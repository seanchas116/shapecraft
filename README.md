# Lattice

Lattice is a 3D modeling tool which enables you to make 3D shapes in the same way as 2D drawing tool.

## MVP Features

* Create 2D shapes
* Create primitive 3D shapes like cube or sphere
* Combine and modify shapes
    * Boolean
    * Extrude
    * Fillet
    * Revolve
    * ...
* Exporting
    * as CAD data (STEP)
    * as mesh (OBJ)

## Future Plans

* Materials

* ...
